# Style Guides

We tend to create living style guides (opens new window) to document our projects’ front-end components. We like to use Storybook (opens new window) or Fractal (opens new window) for this purpose as they provides a lot of features and can be easily customized.

Fractal doesn’t handle assets management on its own, therefor we made our own starterkit (opens new window) that is preconfigured with a couple of tools, especially webpack.

Check out the Liip Styleguide (opens new window) for an example using Fractal.
