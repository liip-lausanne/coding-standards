# Vite

We prefer Vite (opens new window) over other build tools, because:

  • it works out of the box with all our usual file types
  • has a built-in dev server with HMR
  • it’s super fast
  • can be integrated with all type of backends

# Webpack

Older projects, and projects that requires it for whatever reason can use webpack (opens new window) instead.
